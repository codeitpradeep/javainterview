public -<br>
This is the access modifier of the main method <br>
It has to be public so that java runtime can execute this method<br>

static -<br>
In this method has to be static so the JVM can load the class into memory and call the main method<br>

void -<br>
Java main method doesn't return anything, thats why its return type is void<br>

main -<br>
This is the name of java main method, Its fixed and when we run java program, it looks for the main method .<br>
String[] arges. Java main method accepts a single argument of type String array. This is also called asa java command line arguments
