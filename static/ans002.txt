public -
This is the access modifier of the main method 
It has to be public so that java runtime can execute this method

static -
In this method has to be static so the JVM can load the class into memory and call the main method

void -
Java main method doesn't return anything, thats why its return type is void

main -
This is the name of java main method, Its fixed and when we run java program, it looks for the main method
String[] arges. Java main method accepts a single argument of type String array. This is also called asa java command line arguments
