java.util.Scanner- A simple text scanner which can parse primitive types and strings usingregular expressions. 
<br>A Scanner breaks its input into tokens using adelimiter pattern, which by default matches whitespace. The resultingtokens may then be converted into values of different types using thevarious next methods. 
<br>
<br>public int nextInt() - Scans the next token of the input as an int. 
<br>An invocation of this method of the form nextInt() behaves in exactly the same way as theinvocation nextInt(radix), where radixis the default radix of this scanner.
<br>Returns:the int scanned from the input
